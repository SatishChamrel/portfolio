About Me

My name is Satish Chamrel a cybersecurity student with a strong background in network security and operations. I have a experience as a Data Center Technician, where my primary role was to monitored NOC platforms like Zabbix, Nagios, and VMware, troubleshot network and server issues, and configured firewalls, routers, and switches. The expertise includes network traffic analysis using tools like Wireshark and TCPdump, as well as drafting security policies to ensure system accessibility and protection. Proficient in Windows, Linux, and Unix environments and has strong IT help desk experience. I hold multiple certifications, including Fortinet NSE 1 & 2, ICSI (Certified Network Security Specialist), and CNSP (Certified Network Security Practitioner). I have also actively engaged in cybersecurity discussions, having participated in THREAT CON Kathmandu and volunteered as a Network Administrator.
